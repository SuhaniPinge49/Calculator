# Simple Calculator

A web-based calculator that performs basic arithmetic operations like addition, subtraction, multiplication, and division. User-friendly and works on all devices.

## Features
- Add, subtract, multiply, divide numbers
- Simple and clean interface
- Works on desktop and mobile browsers

## How to Use
1. Open `calculator.html` in any web browser.
2. Enter numbers and click the operation buttons to calculate.
